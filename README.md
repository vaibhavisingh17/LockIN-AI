# LockIN-AI 🔒

LockIN-AI is an ML-powered student productivity and burnout prediction platform designed to help students maintain healthy study habits and improve focus.

## Features

- Burnout Prediction using Machine Learning
- Productivity Score Analysis
- AI-generated Study Timetable
- Study vs Sleep Analytics
- Distraction Tracking
- Modern Glassmorphism UI
- Personalized Dashboard
- Session History Tracking

## Tech Stack

- Python
- Flask
- Scikit-learn
- HTML
- CSS
- Bootstrap
- Chart.js
- SQLite

## Machine Learning

The ML model predicts burnout levels using:
- Study Hours
- Sleep Hours
- Mood Levels
- Distraction Count
